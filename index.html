FireOS Distribution Guide

1. Go to the following URL: www.github.com/TfsOs/FireOs/

2. Click on the green "Code" button or the "Raw" button on the GitHub page.

Code Button

3. Next, click on "Download ZIP".

4. After downloading, unzip the file and place it in a folder.

5. Open the folder using a code editor such as VSCode.

6. Make changes as needed, and don't forget to rename the OS to avoid copyright issues.

after this you can do anything you want with your distro
